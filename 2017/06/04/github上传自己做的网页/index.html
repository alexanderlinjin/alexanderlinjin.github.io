    以下均为本人原创

    操作系统：Ubuntu16.04，操作时间：2017.06.04
    在苦恼自己做的网页只能本地查看，但服务器又要钱，又想免费用别人的服务器？
    我来教你自己写的网页想上传到github，并实现网上就能看到自己做的网页。不用犹豫，跟着下面的步骤走就可以了,手速快的一分钟搞定。

    第一步：创建新的仓库

    新建一个仓库,这里举例叫helloworld(截图打少一个l).创建时把这个选项Initialize this repository with a README给勾上

    第二步：找到settings

    进入刚创建的仓库，进入settings,一直往下拉，拉到GitHub Pages
    可以看到这样


    点击master branch,然后点击save。

    然后点击choose a theme,随便选一个主题，点绿色按钮保存



    选完主题后，拉到最下面点击以下commit changes

    github上的操作完成，下面终端操作

    clone你的仓库

    1.打开你的终端，输入git clone git@github.com:XXX/helloworld.git，把刚刚创的仓库克隆到本地，XXX就是你的用户名，helloworld就是你的仓库名。
    2.然后/home/用户/下就多了一个helloworld的文件夹

    3.把你的网页复制进helloworld文件夹中，记住，不要父级目录，一般我们制作网页时会起一个例如叫test的文件夹，然后在里面创建子级文件夹img,css,js等文件，这里不要父级文件夹，即test，不然到时候输入域名打开不了你的网页


    4,提交你的网站

    最后在浏览器输入XXX.github.io/helloworld,成功
